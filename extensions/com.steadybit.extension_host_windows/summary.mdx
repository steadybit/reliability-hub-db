# Introduction

With this extension you are able to discover and attack your Windows hosts in different ways, like modifying network traffic or shutting down processes.

The extension's networking attacks are based on a network driver provided alongside the extension.
The driver is not persistent and is only active during the duration of the attack.
The driver passed all of the [HLK](https://learn.microsoft.com/en-us/windows-hardware/test/hlk/) tests and is signed by Microsoft, assuring safety.

In case you're looking for support of Linux hosts, check out our [Linux Host extension](/extension/com.steadybit.extension_host).
