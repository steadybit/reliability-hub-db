# Introduction

Generate CPU load for one or more cores

# Prerequisites

If you are not using our container images for the extension, you need install the `stress-ng` tool to use the attack.

# Use Cases

* Test your application's resilience to high CPU load

# Parameters

| Parameter | Description                                        | Default   |
|-----------|----------------------------------------------------|-----------|
| CPU Load  | How many CPU  should be consumed?                  | 100       |
| Worker    | How many workers should be used to stress the CPU? | All Cores |
| Duration  | How long should CPU be consumed?                   | 30s       |
