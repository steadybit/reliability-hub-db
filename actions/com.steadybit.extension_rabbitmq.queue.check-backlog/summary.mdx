# Introduction

With this action, Steadybit can check the backlog of a queue for a given time.

# Use Cases

 - You can check the queue backlog during your experiments.
 - You can fail the experiment if the backlog is over a certain threshold.

# Parameters

| Parameter               | Description                                          | Default |
|-------------------------|------------------------------------------------------|---------|
| Duration                | The duration of the attack.                          | 10s     |
| Queue to track backlog  | Required. The queue to check the backlog count.      |         |
| Backlog alert threshold | Required. The acceptable threshold of backlog count. |         |
