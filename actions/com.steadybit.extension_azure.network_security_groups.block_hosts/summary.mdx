# Introduction

This action leverages Azure Network Security Groups to block inbound or outbound traffic to the host. 

# Use Cases

- Understand how the infrastructure with the assigned Network Security Group behaves when specific hosts are blocked. 


# Parameters

| Parameter   | Description                                                                       | Default |
| ----------- | --------------------------------------------------------------------------------- | ------- |
| Duration    | The duration of the attack.                                                       | 30s     |
| Hosts       | Comma-delimited values of the hosts to block                                      |         |
