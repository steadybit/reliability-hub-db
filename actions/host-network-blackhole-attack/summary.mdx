# Introduction

Drops all network traffic (IN/OUT/FORWARDED).

# Prerequisites

If you are not using our container images for the agents, you need install the `tc` (from the iproute2 package) tool to use the attack.

# Use Cases

    * Understand how your services behave under network outages

# Parameters

| Parameter | Description                           | Default |
|-----------|---------------------------------------|---------|
| Duration  | How long should traffic been blocked? | 30s     |
