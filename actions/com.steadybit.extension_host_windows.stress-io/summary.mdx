# Introduction

Generate disk operations on mass storage devices (HDD/SSD).

# Use Cases

* Stress test the host resources

# Parameters


| Parameter           | Description                                                                                                                                   | Default                |
|---------------------|-----------------------------------------------------------------------------------------------------------------------------------------------|------------------------|
| Thread Count        | Define the number of threads for IO operations.                                                                                               | 12                     |
| IO Stress Layer     | On which layer IO is stressed? (File System, Named Partition, Physical Disk)                                                                  | Named Partition        |
| Path                | Path where IO should be inflicted?                                                                                                            | /                      |
| Disable Caching     | Disables both software and hardware write caching                                                                                             | true                   |
| Duration            | How long should IO be consumed?                                                                                                               | 30s                    |
