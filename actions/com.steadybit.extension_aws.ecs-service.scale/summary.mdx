# Introduction

This action will scale a ECS Service to a given desired count of tasks.

# Use Cases

- Check how long it takes to scale your service to a desirable amount of tasks

# Parameters

| Name          | Required | Description                            | Default |
|---------------|----------|----------------------------------------|---------|
| Duration      | true     | How long should the new scale be used? | 180s    |
| Desired Count | true     | The desired task count                 | 1       |
