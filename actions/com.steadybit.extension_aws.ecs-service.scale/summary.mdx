# Introduction

This action will scale a ECS Service to a given desired count of tasks.

# Use Cases

- Check how long it takes to scale your service to a desirable amount of tasks

# Parameters

| Parameter     | Description                            | Default |
|---------------|----------------------------------------|---------|
| Duration      | How long should the new scale be used? | 180s    |
| Desired Count | The desired task count                 | 1       |
