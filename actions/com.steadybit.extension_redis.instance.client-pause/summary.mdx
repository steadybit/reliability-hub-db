# Introduction

With this action, Steadybit can suspend all client command processing on a Redis instance for a given duration using the CLIENT PAUSE command.

# Use Cases

 - Simulate Redis unresponsiveness to test timeout and retry logic.
 - Test read-only mode by pausing only write commands.

# Parameters

| Parameter  | Description                                                                    | Default |
|------------|--------------------------------------------------------------------------------|---------|
| Duration   | How long to pause client connections.                                          | 30s     |
| Pause Mode | Pause scope: "ALL" (all commands) or "WRITE" (write commands only).            | ALL     |
