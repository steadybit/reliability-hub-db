# Introduction

This action stops a process by pid or name.

# Details

The action stops all processes matching the given criteria over the specified duration.

The command `taskkill` is used to stop the targeted processes. If the `graceful` parameter is disabled `taskkill` is executed using `/F` to force the stop.

# Parameters

| Parameter | Description                                                                            | Default |
|-----------|----------------------------------------------------------------------------------------|---------|
| process   | PID or string to match the process name or command                                     |         |
| graceful  | `true` executes a normal stop, `false` a force stop                                    | true    |
| duration  | Over this period the matching processes are killed                                     | 30s     |
| delay     | The initial delay and subsequent interval in which processes are looked up and stopped | 5s      |
