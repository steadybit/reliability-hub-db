# Introduction

When executing chaos experiments, you may want to mute your Instana alerts not to bother your ops colleagues. You can do this with this action.

To create a maintenance window the step can be dragged&dropped into the experiment editor.

# Use Cases

 - Avoid false positives in your monitoring system
 - Avoid alerting your ops colleagues during chaos experiments
 - Avoid to distract your baselines during chaos experiments

# Parameters

| Parameter                       | Description                                    | Default |
| ------------------------------- | ---------------------------------------------- | ------- |
| Duration                        | How long should the maintenance window exist?  | 30s     |
