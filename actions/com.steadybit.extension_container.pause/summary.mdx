# Introduction

Pauses one or more container for a certain time.

# Details

If a container is restarted during the pause, the action will stop (not fail).

_Note:_ Only available for the docker and containerd container runtimes.


# Use Cases

* Interrupting a container for a certain time to simulate a outage

# Parameters

| Parameter | Description                              | Default |
|-----------|------------------------------------------|---------|
| Duration  | How long should the container be paused? | 30s     |
