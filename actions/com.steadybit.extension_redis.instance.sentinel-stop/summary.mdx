# Introduction

With this action, Steadybit can stop a Redis Sentinel server for a given duration using the DEBUG SLEEP command, making it unresponsive to simulate Sentinel failures.

# Use Cases

 - Test failover behavior when a Sentinel becomes unresponsive.
 - Verify that the remaining Sentinels can still achieve quorum and trigger failover.

# Parameters

| Parameter | Description                              | Default |
|-----------|------------------------------------------|---------|
| Duration  | How long Sentinel should be unresponsive.| 30s     |
