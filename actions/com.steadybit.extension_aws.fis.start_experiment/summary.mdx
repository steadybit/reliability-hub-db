# Introduction

Execute AWS FIS Experiments via Steadybit

# Details

With the AWS FIS Experiments, you can easily attack cloud resources without having to install agents on every machine.

# Use Cases

* Run an [AWS FIS Experiment](https://docs.aws.amazon.com/fis/latest/userguide/experiments.html) via Steadybit to combine it with non-FIS or even non-AWS infrastructure

# Rollback

Rollbacks must be defined and supported by the AWS FIS Experiment.


# Parameters

| Parameter | Description                                            | Default   |
| --------- | ------------------------------------------------------ | --------- |
| Duration  | Estimation on how long the experiment will be running? |           |
