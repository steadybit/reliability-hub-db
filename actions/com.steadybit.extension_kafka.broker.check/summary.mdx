# Introduction

With this action, Steadybit can check the activity of brokers for a given time.

# Use Cases

 - You can check the activity of your brokers during your experiments (broker downtime and controller elections).

# Parameters

| Parameter         | Description                                                               | Default |
|-------------------|---------------------------------------------------------------------------|---------|
| Duration          | The duration of the attack.                                               | 10s     |
| Expected Changes  | The changes to look after (Broker Downtime, New Controller Elected)       |         |
| Change Check Mode | How often should the state be expected. "At least once" or "All the time" |         |
