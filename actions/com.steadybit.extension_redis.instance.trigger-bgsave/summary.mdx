# Introduction

With this action, Steadybit can trigger a Redis BGSAVE command to initiate a background RDB snapshot on a Redis instance. This is an instantaneous action with no parameters.

# Use Cases

 - Test application performance during background save operations.
 - Verify that BGSAVE on large datasets doesn't cause excessive latency.

# Parameters

This action has no parameters. It triggers an instantaneous BGSAVE command.
