# Introduction

Inject corrupt packets by introducing a single bit error at a random offset into outgoing network traffic.

# Prerequisites

The Extension Windows Host needs to be installed on the target system.

# Use Cases

* Testing network resilience to bit errors

# Parameters

| Parameter          | Description                                                                       | Default |
|--------------------|-----------------------------------------------------------------------------------|---------|
| Duration           | How much of the traffic should be corrupted??                                     | 30s     |
| Package Corruption | How much traffic should be corrupted?                                             | 15%     |
| Hostnames          | (optional) Restrict to/from which hosts the traffic is affected.                  |         |
| IPs/CIDRs          | (optional) Restrict to/from which IP addresses or blocks the traffic is affected. |         |
| Ports              | (optional) Restrict to/from which ports the traffic is affected.                  |         |
| Network Interface  | (optional) Target Network Interface which should be attacked                      |         |
