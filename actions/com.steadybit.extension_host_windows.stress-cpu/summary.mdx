# Introduction

Generate CPU load for one or more cores

# Use Cases

* Test your application's resilience to high CPU load

# Parameters

| Parameter        | Description                                                    | Default   |
|------------------|----------------------------------------------------------------|-----------|
| CPU Load         | How much CPU should be consumed?                               | 100       |
| Cores            | How many CPU cores should be targeted during the stress attack?| All Cores |
| Process Priority | What is the priority of the stress process?                    | Normal    |
| Duration         | How long should CPU be stressed?                               | 30s       |
