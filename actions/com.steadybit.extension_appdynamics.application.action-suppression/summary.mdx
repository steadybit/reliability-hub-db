# Introduction

The Create Action Suppression step can be used to disable actions related to events in AppDynamics for a given duration.
You can use it during your experiments to disable any unwanted automatic actions because of your chaos engineering experiments.

# Parameters

| Parameter                      | Description                                               | Default |
|--------------------------------|-----------------------------------------------------------|---------|
| Duration                       | How long should the state of the alert rule be checked    | 30s     |
| Disable Agent Metric Reporting | Disable agent metric reporting during the given duration. | false   |
