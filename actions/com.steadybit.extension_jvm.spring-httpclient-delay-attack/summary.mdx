# Introduction

Delays a response from a Spring `RestTemplate` or `WebClient` by the given duration.

# Details

This attack only targets JVMs identified as Spring Application.

# Use Cases

* Understand how your services behave under slow network connections to other services

# Parameters

| Parameter    | Description                                                                                           | Default |
|--------------|-------------------------------------------------------------------------------------------------------|---------|
| Duration     | How long should the response be delayed?                                                              | 30s     |
| Delay        | How much should the response be delayed?                                                              | 500ms   |
| Jitter       | (optional) Random +-30% jitter to response delay                                                      | false   |
| Host Address | (optional) Which host address should be attacked                                                      | *       |
| Path Pattern | (optional) Which URL path should be attacked? Supports Spring Ant-style path patterns (e.g. /api/**). | /**     |
| Http Methods | (optional) Which HTTP methods should be attacked?                                                     |         |
