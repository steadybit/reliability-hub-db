# Introduction

With this action, Steadybit can check the state of a consumer for a given time.

# Use Cases

 - You can check the state of your consumer during your experiments.

# Parameters

| Parameter           | Description                                                                                               | Default |
|---------------------|-----------------------------------------------------------------------------------------------------------|---------|
| Duration            | The duration of the attack.                                                                               | 10s     |
| Expected State List | Required. The state to look after (Dead, Stable, Empty, Unknown, CompletingRebalance, PreparingRebalance) |         |
| State Check Mode    | How often should the state be expected. "At least once" or "All the time"                                 |         |
