# Introduction

Run a Jenkins Build job as part of your Steadybit Experiment.

# Parameters

| Parameter           | Description                                           | Default |
|---------------------|-------------------------------------------------------|---------|
| Estimated Duration  | Estimation on how long the build job will be running. | 60s     |
| Wait for Completion | Should the step wait for the job to complete?         | true    |
| Parameters          | (optional) Job Parameters                             |         |
