# Introduction

With this action, Steadybit can check the activity of nodes for a given time.

# Use Cases

 - You can check the activity of your nodes during your experiments (node downtime and node alarms).

# Parameters

| Parameter         | Description                                                               | Default |
|-------------------|---------------------------------------------------------------------------|---------|
| Duration          | The duration of the attack.                                               | 10s     |
| Expected Changes  | The changes to look after (Node Downtime, Node Alarms)                    |         |
| Change Check Mode | How often should the state be expected. "At least once" or "All the time" |         |
