# Introduction

[Gatling](https://gatling.io/) is a powerful load-testing solution for applications, APIs, and microservices. You can integrate the execution of Gatling load tests directly into your experiments.

# Details

This action allows you to run existing Gatling tests in your Gatling Enterprise account. It requires a valid Gatling Enterprise account and the corresponding API key.

# Parameters

| Parameter              | Description                                                                                                                                                 | Default |
|------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------|---------|
| Estimated duration     | The step will run as long as needed. You can set this estimation to size the step in the experiment editor for a better understanding of the time schedule. | 30s     |
| Java System Properties | (optional) Java System Properties passed to the simulation                                                                                                  |         |
| Environment variables  | (optional) Environment variables passed to the simulation                                                                                                   |         |
