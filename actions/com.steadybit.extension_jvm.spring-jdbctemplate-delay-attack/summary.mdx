# Introduction

Delay a Spring JDBC Template response by the given duration.

# Details

This attack only target JVMs identified as Spring Application.

# Parameters

| Parameter           | Description                                          | Default |
|---------------------|------------------------------------------------------|---------|
| Operation           | Which operation should be attacked?                  | *       |
| Duration            | How long should the traffic be delayed?              | 30s     |
| Delay               | How much should the traffic be delayed?              | 500ms   |
| Jitter              | (optional) Random +-30% jitter to network delay      | false   |
| JDBC connection url | (optional) Which JDBC connection should be attacked? | *       |
